# SahAIbat × YPPS Partnership Presentation

Interactive 10-slide presentation prepared for the YPPS (Yayasan Pengkajian dan Pengembangan Sosial) leadership meeting.

Bahasa Indonesia primary, with English annotations throughout.

## Contents

- **Slide 1**: Cover
- **Slide 2**: Why this meeting — grant competition context
- **Slide 3**: What is SahAIbat — four products overview
- **Slide 4**: Integration model — how we add a layer, not replace
- **Slide 5**: TAKENUSA program fit (maternal / neonatal triage)
- **Slide 6**: SALAM program fit (SARI public chatbot)
- **Slide 7**: Value proposition
- **Slide 8**: Donor opportunities — Wellcome, Gates, USAID, Kemenkes, Oxfam
- **Slide 9**: Engagement model — 30-minute onboarding, no big training
- **Slide 10**: Next steps

## Navigation

- Arrow keys (← →), Page Up / Page Down, Space
- Click navigation buttons at the bottom
- Click any dot to jump to a slide
- Swipe left/right on mobile

## Local Development

This is a static HTML file. No build step required. Simply open `index.html` in a browser.
